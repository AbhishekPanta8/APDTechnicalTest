# Weather Forecast Frontend

## Overview

React-based frontend application that displays weather forecast data in a clean, responsive interface.

## Components

- **WeatherDashboard**: Main container component
- **WeatherCard**: Individual day forecast card
- **LoadingSpinner**: Loading state indicator
- **ErrorMessage**: Error display component

## Services

- **weatherApi**: Axios-based API client for backend communication

## Utilities

- **formatters**: Temperature and date formatting functions

## Key Features

- Responsive design
- Loading states
- Error handling with retry mechanism
- Clean, minimal UI
- Real-time data fetching

## Running

Development mode:
```bash
npm start
```

Build for production:
```bash
npm run build
```
